# Mini Numpy Rust
Every day at my job, I use a whole host of Python libraries (PyTorch, NumPy, etc) that have "C++ bindings". While I've always understood roughly what that meant, I've never actually spent any time exploring what that means, and was always somewhat curious.
Given that Rust is constantly gaining traction in the ML world, I figured I would try implementing a bare bones variant of NumPy based on Rust as an educational exercise.
